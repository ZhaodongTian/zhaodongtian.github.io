    Remarks

    If the <xref:CoreWCF.Security.X509ClientCertificateAuthentication.CertificateValidationMode%2A> is set to <xref:CoreWCF.Security.X509CertificateValidationMode.Custom>, then you must create a custom certificate validator that inherits from the <xref:System.IdentityModel.Selectors.X509CertificateValidator> class. For more information, see How to: Create a Service that Employs a Custom Certificate Validator.
