    Remarks

    After the <xref:System.IdentityModel.Claims.ClaimSet> class has been created, it cannot be changed.

    Examining the content of a <xref:System.IdentityModel.Claims.ClaimSet> for particular types of claims is a common task when using claim-based authorization. To examine a <xref:System.IdentityModel.Claims.ClaimSet> for the presence of particular claims, use the <xref:System.IdentityModel.Claims.ClaimSet.FindClaims%2A> method. The method provides better performance than iterating directly over the <xref:System.IdentityModel.Claims.ClaimSet>.
