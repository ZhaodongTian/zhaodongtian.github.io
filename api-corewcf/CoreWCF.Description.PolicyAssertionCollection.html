    public class PolicyAssertionCollection : Collection<XmlElement>
    Remarks

    Use the <xref:CoreWCF.Description.PolicyAssertionCollection> class to group together policy assertions and to enumerate or modify a group of assertions.

    The <xref:CoreWCF.Description.PolicyAssertionCollection> is used most frequently as a return value for the methods of the <xref:CoreWCF.Description.PolicyConversionContext> class.
