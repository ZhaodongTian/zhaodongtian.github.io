    Remarks

    When using a WCF REST endpoint (WebHttpBinding and WebHttpBehavior or <xref:CoreWCF.Description.WebScriptEnablingBehavior>) the HTTP status code on the response is set accordingly. However, WebFaultException can be used with non-REST endpoints and behaves like a regular <xref:CoreWCF.FaultException>.

    Use this class to return error details using the same format as the operation.
