    Class WebHttpDispatchOperationSelector

    Remarks

    The <xref:CoreWCF.Dispatcher.WebHttpDispatchOperationSelector> dispatches method calls based on matching the Uniform Resource Identifier (URI) to a <xref:System.UriTemplateTable>. If multiple matches are found in the <xref:System.UriTemplateTable>, dispatch is done by the HTTP method.
