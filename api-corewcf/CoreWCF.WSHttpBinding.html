    Remarks

    The <xref:CoreWCF.WSHttpBinding> is similar to the <xref:CoreWCF.BasicHttpBinding> but provides more Web service features. It uses the HTTP transport and provides message security, as does <xref:CoreWCF.BasicHttpBinding>, but it also provides transactions, reliable messaging, and WS-Addressing, either enabled by default or available through a single control setting.
