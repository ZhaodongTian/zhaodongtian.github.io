    Class EndpointAddress10

    Provides a serializable type of endpoint address that is WS-Addressing V1 compliant and that enables it to be exposed as part of the service contract.

    Remarks

    The <xref:CoreWCF.EndpointAddress> is not serializable nor is it tied to a particular version of the WS-Addressing specification. This class provides a WS-Addressing V1 compliant wrapper for the <xref:CoreWCF.EndpointAddress> that is serializable (implements the <xref:System.Runtime.Serialization.ISerializable> interface) and bound to the WS-Addressing V1 wire format. The <xref:CoreWCF.EndpointAddress10.FromEndpointAddress%28CoreWCF.EndpointAddress%29> method does the wrapping and the <xref:CoreWCF.EndpointAddress10.ToEndpointAddress%2A> method does the unwrapping, allowing the recovery of the endpoint address after it is serialized on the wire.

    Windows Communication Foundation (WCF) also supplies a class that provides a serializable type, <xref:CoreWCF.EndpointAddressAugust2004>, which is WS-Addressing August 2004 compliant for legacy purposes.
