    Class EndpointIdentity

    An abstract class that when implemented provides an identity that enables the authentication of an endpoint by clients that exchange messages with it.

    Remarks

    The value of the identity for an endpoint can be set with constructors of <xref:CoreWCF.EndpointAddress>, where its value can also be accessed as a read-only <xref:CoreWCF.EndpointAddress.Identity%2A> property.
