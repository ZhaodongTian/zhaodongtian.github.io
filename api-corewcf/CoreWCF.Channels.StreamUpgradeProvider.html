    Remarks

    An example of a stream upgrade is to build a compression stream directly on top of the transport stream. To add a security upgrade, you additionally implement a class that derives from <xref:CoreWCF.Channels.StreamSecurityUpgradeProvider>. You can implement multiple upgrades, each wrapping the preceding one.

    The upgrade process has the following steps:

    1. Implement a class that derives from <xref:CoreWCF.Channels.StreamUpgradeProvider>.

    2. Register it. You do this by creating a binding element for it, and registering the provider in a custom binding.

    3. The derived class that you implement creates an instance of <xref:CoreWCF.Channels.StreamUpgradeInitiator>, which at run time initiates a request to the other end of its connection to upgrade the channel transport layer.

    4. The derived class that you implement creates an instance of <xref:CoreWCF.Channels.StreamUpgradeAcceptor>, which at run time receives the upgrade request from the preceding step and, if possible, accepts the upgrade and returns the upgraded stream to the other side of its connection.

    5. If there are additional upgrade requests, they are processed sequentially, as described in the preceding two steps.
