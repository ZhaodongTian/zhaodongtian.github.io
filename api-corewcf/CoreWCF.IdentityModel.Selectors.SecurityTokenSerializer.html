    Class SecurityTokenSerializer

    SecurityTokenSerializer is responsible for writing and reading SecurityKeyIdentifiers, SecurityKeyIdentifierClauses and SecurityTokens. In order to read SecurityTokens the SecurityTokenSerializer may need to resolve token references using the SecurityTokenResolvers that get passed in. The SecurityTokenSerializer is stateless Exceptions: XmlException, SecurityTokenException, NotSupportedException, InvalidOperationException, ArgumentException
