    Remarks

    The <xref:CoreWCF.Description.MessagePropertyDescription> class represents message properties specified in a message contract by the use of the <xref:CoreWCF.MessagePropertyAttribute?displayProperty=nameWithType>. Message properties are used to place information on the message that can be used by other objects further on in the message pipeline but that are not serialized with the message.
