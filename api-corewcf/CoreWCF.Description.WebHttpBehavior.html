    Class WebHttpBehavior

    Enables the Web programming model for a Windows Communication Foundation (WCF) service.

    Inheritance
    Object
    WebHttpBehavior
    Implements
    IEndpointBehavior
    Namespace: CoreWCF.Description
    Assembly: CoreWCF.WebHttp.dll
    Syntax
    public class WebHttpBehavior : Object, IEndpointBehavior
    Remarks

    The <xref:CoreWCF.Description.WebHttpBehavior> behavior, when used in conjunction with the <xref:CoreWCF.WebHttpBinding> binding, enables WCF to expose and access Web style services. <xref:CoreWCF.Web.WebServiceHost> automatically adds this behavior to endpoints that use the <xref:CoreWCF.WebHttpBinding>.
